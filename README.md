# LeetPulse

LeetPulse is an AI-powered LeetCode analytics and coaching platform that analyzes user performance and generates personalized insights using LLMs.

## Features
- LeetCode performance analytics by difficulty and topic
- Interactive dashboards (Next.js + Recharts)
- LLM-powered insights: strengths, weaknesses, study plans
- Always-on backend during interview demos

## Tech Stack
**Frontend**
- Next.js (TypeScript)
- Tailwind CSS
- Recharts

**Backend**
- FastAPI
- Python
- LLM-based insight generation
